3D Rubik's Cube Viewer

The 3D Rubik's Cube Viewer is a captivating web-based application that brings the iconic Rubik's Cube to life in a virtual space. Built using HTML, CSS, and JavaScript, this project utilizes the powerful Three.js library to render a realistic 3D representation of the cube. Users can enjoy a seamless and interactive experience as they rotate and manipulate the cube using mouse actions. Each face of the cube features colored stickers representing different colors, just like the real thing. The application also offers an exciting challenge, allowing users to solve the cube by rotating its faces strategically. This project demonstrates my proficiency in HTML, CSS, and JavaScript for web development, as well as my understanding of 3D graphics and rendering techniques using Three.js. By creating an engaging and visually appealing web application, the 3D Rubik's Cube Viewer serves as a testament to my ability to implement complex interactions and animations that captivate users and make a positive impact on my portfolio.
